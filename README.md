## Assigment-2:
**Purpose:**
This part of the assignment was to build an AI-based review analysis assistant that processes customer reviews and returns structured insights using a predefined schema.

**Working of assignment:**
User Review -> Python Application -> LangChain -> Ollama -> Phi-3 LLM -> Pydantic Output Parser -> Structured JSON Response -> Displayed to User

**Technologies used:**
1. **LangChain** – Used to interact with the LLM and manage prompt formatting.
2. **Pydantic** – Used to define the structured schema for the LLM output.
3. **Pydantic Output Parser** – Ensures the LLM response follows the predefined schema.
4. **Ollama** – Allows running Large Language Models locally without external APIs.
5. **Phi-3 Mini Large Language Model** – An open-source LLM used to analyze the customer reviews.

**Work Summary:**

- In this assignment, I developed a review analysis assistant that processes customer feedback and extracts structured insights. The application accepts a customer review as input and sends it to a Large Language Model running locally through Ollama.
- A **Pydantic schema** is used to define the required structure of the output, including the review summary, positive aspects, negative aspects, and overall sentiment.
- The **Pydantic Output Parser** ensures that the response generated by the model strictly follows this schema.
- The output is returned in a structured JSON format so that it can be easily interpreted or integrated into other systems.

## Assigment-3:
**Purpose:**
This part of the assignment was to build a conversational AI assistant that can interact with users while maintaining chat history. 
The goal was to ensure that the assistant remembers previous messages and uses that context to generate meaningful responses in multi-turn conversations.

**Working of assignment:**
User Message -> Python Application -> Chat History Storage -> Ollama -> Phi-3 LLM -> Context-Aware Response -> Displayed to User

**Technologies used:**
1. **LangChain** – Used to interact with the language model and manage chat messages.
2. **Ollama** – Allows running Large Language Models locally on the system.
3. **Phi-3 Mini Large Language Model** – An open-source lightweight LLM used to generate conversational responses.

**Work Summary:**
- In this assignment, I developed a conversational AI assistant that maintains conversation history during interactions with the user.
- The assistant accepts user messages through the terminal and stores each message in a chat history list.
- Each time the user sends a new message, the complete conversation history is sent to the LLM through Ollama.
- This allows the model to understand the context of the conversation and generate responses that are relevant to earlier messages.
- By maintaining a list of chat messages, the assistant can handle follow-up questions and multi-turn conversations effectively.

## Assignment-4:
**Purpose:**
This part of the assignment was to build an AI-powered Resume Reviewer that evaluates candidate resumes for a specific job role using Prompt Engineering. 
The goal was to guide a Large Language Model to analyze resumes and generate structured feedback about the candidate’s suitability for the role.

**Working of assignment:**
Resume Input + Job Role -> Python Application -> Prompt Engineering -> Ollama -> Phi-3 LLM -> Structured Resume Evaluation -> Displayed to User

**Technologies used:**
1. **LangChain** – Used to interact with the language model and manage prompt templates.
2. **Ollama** – Allows running Large Language Models locally without external APIs.
3. **Phi-3 Mini Large Language Model** – An open-source lightweight LLM used to analyze resumes and generate evaluations.

**Work Summary:**

- In this assignment, I developed an AI Resume Reviewer that analyzes candidate resumes for a specific job role using Prompt Engineering techniques.
- The system accepts two inputs from the user: the resume text and the job role.
- A carefully designed prompt instructs the Large Language Model to behave like an experienced technical recruiter.
- The prompt guides the model to analyze the candidate’s skills, experience, and education in relation to the job role requirements.
- Based on this analysis, the model generates structured feedback including a candidate summary, matching skills, missing skills, experience evaluation, and an overall recommendation about the candidate’s suitability for the role.
